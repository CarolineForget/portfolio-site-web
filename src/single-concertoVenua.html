Concerto Venua

Voir le site web

Résumé

Concerto Venua est un festival (fictif) de musique classique réinventé se déroulant dans la magnifique ville de Dublin. Il a pour but d’intéresser les jeunes adultes au style musical classique.

C’est un projet scolaire réalisé en hiver 2020 avec les logiciels Figma et VS Code. J’ai fait le design UI/UX et l'intégration web (HTML/CSS/JS).

  • Réalisé par : Caroline Forget
  • Images de : Unsplash

Wireframes

Voir sur Figma

J’ai réalisé des wireframes en version desktop pour optimiser l’expérience de l’utilisateur avec le site web en organisant et structurant les interfaces.

Direction artistique

Dans le but d’harmoniser les côtes moderne et classique de Concerto Venua, j’ai choisi une palette sobre avec des couleurs modernes. L’utilisation de typographies Serif, Sans Serif et manuscrite représente l’harmonisation de ces deux réalités.
